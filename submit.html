Assignment 3

Assignment Question:
You are tasked with creating a Python program that simulates a simple calculator. Your program should include the following:
1. A function named "addition" that takes two numbers as input and returns their sum.
2. A function named "subtraction" that takes two numbers as input and returns their difference.
3. A function named "multiplication" that takes two numbers as input and returns their product.
4. A function named "division" that takes two numbers as input and returns their quotient.
5. A module named "calculator" that imports the above functions and provides an interactive menu to the user.
Write a Python program that implements the above requirements. The program should display a menu with the following options:
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Quit
When the user selects an option, the program should prompt for the required inputs, call the corresponding function, and display the result.
Your program should handle input validation, such as checking for invalid menu options and division by zero errors. Additionally, it should continue to display the menu until the user chooses to quit.
Write your program and provide the necessary functions and module to fulfill the requirements of this assignment.
